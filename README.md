## Vue project 
This project it's a web site where the clients can mount the pizza like they want

### How to...

#### Install node [Ubuntu]
```bash
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
```

```bash
sudo apt-get install -y nodejs
```

#### Install Vue CLI
```bash
npm install -g @vue/cli
```

#### Execute project
```bash
npm run serve
```
